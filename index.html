JSON

Published on November 1, 2018

JSON stands for JavaScript Object Notation, it is a text that written with JavaScript. Also, it is a syntax of braces, brackets, colons, and commas that store and replace data between mobile apps and hosted services on the web browser. It is beneficially and not hard to learn, also it is easy to write and read it. This tutorial includes the main topics related to JSON such as, JSON vs XML, Data Types, Objects, JSON Stringify & Parse and JSON HTML. Furthermore, the tutorial aims to explain JSON to help in developing the webpages, so it will has some examples of using JSON with HTML. In the beginning of 1996, it has been discovered by Crockford that it is started to use at Netscape. However according to resource, Douglas Crockford was the first who specify it in 2001. JSON is used to let the developers interchange formats of data on internet in simpler way.
