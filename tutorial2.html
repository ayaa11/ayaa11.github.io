Tutorial 2: JSON Data Types

Published on November 15, 2018

JSON is created to two of data types; primitive and structure or complex data types. Primitive are four data types; string, number, Boolean and null. Structure or call it complex data types are Object and Array. JavaScript objects and arrays can be used as JSON.

Data types in JSON C# Corner

Description and example of each JSON Data Type:

These data types contains each object that has number of values. Also, each value has a pair of key values and this key comes first after that the value follow a colon to print it. JSON array is a list of values but JSON object it a collection pairs values of name.

String: A string written inside quotation marks. A backslash (\) defined as the escape character. This example present the simple array that has two values which are strings.

Number: A number can be an integer or a floating point it depends on implementation. Here an example of the array can have a value like a number.

Boolean: It contains true or false values.

Null: It means an empty values.

Array: array of values.

  • It represents an ordered collection of values.
  • It is begins with square brackets.
  • Value is separated by a comma (,).
  • Values of arrays begin with any valid data type.
  • Contains multiple objects.

This example is more complex to have multiple objects.

Object: array of key/value pairs.

  • It represents an ordered collection of values.
  • It is begins with square brackets.
  • Value is separated by a comma (,).
  • Values of arrays begin with any valid data type.
